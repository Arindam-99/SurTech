Commonly Used HTML Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a>: Anchor (Link)
  • <ul>, <ol>, <li>: Lists
  • <table>, <tr>, <td>, <th>: Tables
  • <img>, <video>: Media Elements
  • <form>, <input>, <textarea>, <select>: Forms
  • <div>, <span>: Generic Containers
  • <header>, <main>, <footer>, <section>: Semantic Tags
  • <br>, <hr>: Line and Section Breaks

HTML Attributes Used

  • href: Link destination (in <a>)
  • src: Source for image/video
  • controls: Show video/audio controls
  • autoplay: Autoplay video
  • loop: Repeat media
  • width, height: Set element size
  • type: Input type in forms
  • required: Make field mandatory
  • action: Backend submission path for forms
  • alt: Text for images
  • target="_blank": Open link in new tab
